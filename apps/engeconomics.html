Glossary ▾
NSR (Net Standard Revenue) = Σ (Hours × NSR Rate/hr). The standard value of resources deployed.

ANSR (Adjusted Net Standard Revenue) = NSR × (1 + EAF%). Revenue recognized from time charges.

EAF (Engagement Adjustment Factor) = (ANSR ÷ NSR) − 1. Premium or discount vs standard rates.

TER (Total Engagement Revenue) = ANSR + Billable Expenses.

Margin = ANSR − (Labour Cost + Tech Fee + Expenses).

NUI (Net Unbilled Inventory) = (Recognized ANSR + Charged Expenses) − (Billed Fees + Billed Expenses).
  NUI > 0 → work done but not yet billed
  NUI = 0 → billed exactly what has been recognized
  NUI < 0 → billed ahead of work performed

Cost Allowance = Fee × (1 − Target Margin%).
