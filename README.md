# Finance Calculators

Standalone HTML calculators and modelers. No build step, no dependencies — just drop files and go.

## How it works

1. Add any `.html` file to `apps/`
2. Push to `main`
3. A GitHub Action scans `apps/`, extracts `<title>` and `<meta name="description">` from each file, and writes `apps/manifest.json`
4. The landing page (`index.html`) fetches that manifest and renders the app list

The index page updates automatically every time you push a new calculator.

## Adding a new calculator

Create an HTML file in `apps/` with at minimum a `<title>` tag. Optionally add a meta description for a subtitle on the landing page:

```html
<title>My Calculator</title>
<meta name="description" content="Short description shown on the index page.">
```

If your calculator defines a `--accent` CSS variable, the landing page icon picks up that color.

## Hosting

Served via GitHub Pages from the `main` branch. No server-side code needed.

## Structure

```
index.html              Landing page (dynamic, reads manifest.json)
apps/                   Calculator HTML files
apps/manifest.json      Auto-generated by GitHub Action
src/                    Prompt/skill source docs
```
